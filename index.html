The Hacker NewsSecurity

Chaos ransomware's new msaRAT backdoor hides C2 traffic inside Chrome/Edge

Cisco Talos disclosed a new Rust-based RAT used by the Chaos ransomware group that launches a headless Chrome or Edge instance and routes command-and-control traffic through the browser via the Chrome DevTools Protocol and WebRTC. This defeats standard network-based detection since defenders only see normal-looking browser traffic.
BleepingComputerSecurity

"RefluXFS" Linux kernel flaw lets local users gain root via XFS race condition (CVE-2026-64600)

Qualys disclosed a race condition in the Linux kernel's XFS filesystem that lets a local user exploit concurrent O_DIRECT writes on a reflink-enabled volume to gain full root access, even under SELinux Enforcing. Qualys estimates over 16.4 million systems (RHEL, Oracle Linux, Amazon Linux, Fedora Server) are potentially affected; the fix merged into the kernel on July 16.
